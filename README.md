# Mistica Web - Original Website 2005

This is the original Mistica band website from 2005, preserved and deployed for historical purposes.

## Structure

The website uses HTML framesets and is located in the `Mistica New Generation` directory.

## Main Entry Points

- `index.htm` - Landing page
- `indexReal.htm` - Main frameset page (left menu + right content)
- `main.htm` - Home page content
- `menu.htm` - Navigation menu

## Deployment

This site is deployed to Netlify and accessible at:
https://joseprendergast.com/Mistica_original_Web_2005

## Local Development

To run locally:

```bash
cd "Mistica New Generation"
python3 -m http.server 8000
```

Then open http://localhost:8000/indexReal.htm in your browser.
